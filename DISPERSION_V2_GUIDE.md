## 📊 如何读图

### 1. 椭圆（中央的彩色区域）

**含义**: PM2.5的影响范围

| 颜色 | PM2.5范围 | 解读 |
|------|----------|------|
| 🟢 浅绿 | <25 | 安全，椭圆几乎消失 |
| 🟠 暗橙 | 25-35 | 中等污染，开始出现 |
| 🟣 橙红 | 35-50 | 高污染，明显椭圆 |
| 🔴 朱红 | 50-100 | 严重污染，大椭圆 |
| 🟥 深红 | ≥100 | 严重，最大椭圆（只有7月24日） |

**大小**:
- 绿色 → 椭圆小（安全范围）
- 红色 → 椭圆中等
- 深红 → 椭圆最大（134 ug/m3）

**形状**:
- 圆形 = 弱风，烟雾四散
- 椭圆 = 风强，烟雾被吹向一个方向

### 2. 风向箭头（从蓝点出发）

**颜色**:
- 🔵 蓝色箭头 = 弱风 (<2 m/s)
- 🟢 绿色箭头 = 轻风 (2-3 m/s)
- 🟠 橙色箭头 = 中等风 (3-4 m/s)
- 🔴 红色箭头 = 强风 (>4 m/s)

**方向**:
- 箭头指向 = 风吹向的方向
- 箭头大小 = 风速强度

**解读**:
- 箭头向西北 + 大椭圆 = "烟雾被西北风吹向卡尔加里东南"
- 箭头向东北 + 小椭圆 = "弱东北风，烟雾轻微影响"

### 3. 蓝点
- 位置: 卡尔加里市中心 (51.0447°N, 114.0719°W)
- 意义: PM2.5监测点
- 用途: 风向箭头和椭圆都以此为中心计算

### 4. 右下数据表

**列说明**:
- **Date**: 日期
- **PM2.5**: 该日PM2.5浓度（ug/m3）
- **Wind**: 风速（m/s）
- **Dir**: 风向（0-360°，0=北，90=东，180=南，270=西）

**粉色背景行**: 峰值污染日

### 打开地图
```
1. 在浏览器中打开 pm25_dispersion_index.html
2. 点击 "Event #1" 或 "Event #2" 链接
3. 等待地图加载完成（约5-10秒）
```

### 观察椭圆变化
```
查看8天的日期：
- 日期1-2（前3天）: 椭圆很小，颜色浅
- 日期3（峰值）: 椭圆最大，颜色最深
- 日期4-8（后4天）: 椭圆逐日缩小，颜色变浅
```

### 观察风向变化
```
8天中的箭头方向：
- 有的日子指向东北→椭圆跟着指向东北
- 有的日子指向西南→椭圆跟着指向西南
这说明风向在变化，污染传输方向在变化
```

### 对比两个事件
```
Event #1 (134 ug/m3):
- 椭圆非常大（100+ km走径）
- 最大的深红色区域
- 明显风向控制扩散方向

Event #2 (53.5 ug/m3):
- 椭圆中等大小（50-70 km）
- 橙红颜色，不如Event #1严重
- 可能的风速、风向不同
```

## 📍 风向度数说明

```
    北(0°/360°)
      ↑
西(270°) → 东(90°)
      ↓
    南(180°)

45° = 东北方        225° = 西南方
135° = 东南方       315° = 西北方
```

**例**：
- 风向45° = 从东北吹来，烟雾被吹向西南
- 风向225° = 从西南吹来，烟雾被吹向东北

## 🔍 技术细节

### 椭圆半径计算
```
基础半径 = 50 km
强度因子 = PM2.5 / 25（阈值）
最终半径 = 基础 × 强度 × 风速因子
范围 = 25-250 km

例子：
- 10 ug/m3 → 25 km 半径
- 50 ug/m3 → 100 km 半径
- 134 ug/m3 → ~200+ km (限制到250)
```

### 椭圆形状
```
风弱 → 圆形（各方向均等扩散）
风强 → 椭圆（风向方向拉长）

长轴 = 半径 × (1 + 0.5 × 风速因子)
短轴 = 半径 × (1 - 0.3 × 风速因子)
```

### 风向箭头
```
颜色: 风速大小
大小: 风速强度
方向: 风吹向的方向
```

## ⚠️ 限制条件

1. **单点模型**: 以Calgary市中心为参考，实际扩散更复杂
2. **简化风场**: 假设均匀风向，实际可能旋转变化
3. **椭圆近似**: 由64个点组成，边界平滑但非完美
4. **粗糙计算**: 速度优先，一些物理过程简化
