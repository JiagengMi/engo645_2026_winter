## 📏 扩散模型说明

### 椭圆参数计算

#### 1. 影响半径（Influence Radius）
```
基础半径 = 50 km
强度因子 = PM2.5 / 25 µg/m³ 阈值 （范围：0.5-2.0倍）
最终半径 = 基础半径 × 强度因子
```

**例子**：
- PM2.5 = 10 µg/m³ → 因子0.4 → 半径 20 km
- PM2.5 = 25 µg/m³ → 因子1.0 → 半径 50 km
- PM2.5 = 50 µg/m³ → 因子2.0 → 半径 100 km
- PM2.5 = 134 µg/m³ → 因子2.0 → 半径 100 km（上限）

#### 2. 椭圆形状（受风速影响）
```
长轴 = 半径 × (1.0 + 0.5 × 风速归一化)
短轴 = 半径 × (1.0 - 0.3 × 风速归一化)
```

**风向含义**：
- 风速强 → 椭圆更细长（烟雾被吹得更直）
- 风速弱 → 椭圆更接近圆形（烟雾扩散）

#### 3. 方向
```
椭圆指向 = 风吹来的反方向
（例：风从北吹来，椭圆指向南）
```

### 几何生成
- 使用参数化椭圆方程
- 旋转到风向

## 📊 示例说明

### "为什么7月24日椭圆最大？"

```
PM2.5 = 134 µg/m³（历史最严重）
风速 = 2.53 m/s（适中）
风向 = 45°（西北风，吹向東南Calgary）

计算：
- 强度因子 = 134/25 = 5.36 → 限制到2.0
- 半径 = 50 × 2.0 = 100 km
- 椭圆方向 = 45° + 180° = 225°（指向西北）
- 长轴 = 100 × (1 + 0.5×2.53/5) = 125 km
- 短轴 = 100 × (1 - 0.3×2.53/5) = 85 km

结果：155大的椭圆，指向西北方向，全是深红色
```

## ⚠️ 限制条件

1. **单点模型**：使用Calgary市中心为参考点
2. **简化风场**：假设均匀风场，实际可能更复杂
3. **粗糙椭圆**：32个点的多边形，实际扩散是渐变的
4. **截断半径**：PM2.5倍增限制在0.5-2.0倍（避免极端值）
